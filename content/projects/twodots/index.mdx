<h2>Game Summary</h2>

Two Dots is a puzzle game that's free to play and loved by millions.
Travel through gorgeous worlds, collect beautiful collectibles, and uncover fun new mechanics along the way. Solve challenging puzzles by connecting the dots and more.

<h2>My Part</h2>

<div style="text-align: left">

Building upon the popular mobile game TwoDots, creating new features, minigames, and puzzle content to steadily increase the game’s player base.
- Architected large features with technical specification docs.
- Responsible for the on time delivery of systems being built by a dev team.
- Worked with the product team on timelines and resource allocation planning for large features.
- One of the lead engineers on TwoDots’ social systems, allowing users to connect with each other.
- Collaborated with the backend team to design and implement a generic inventory system.
- Implemented TwoDots’ daily quest system that can track events from any piece of the game.
- Analyzed and improved app performance, App Start Time, optimized how much local storage TwoDots uses.
- Built Unity editor tools to help automate Designer and QA workflows.
- Maintaining and upgrading a 5+ year old code base.

</div>
